# Spotify Backend API 🎵

This is my first backend project built while learning backend development with Node.js and Express.js. The project is inspired by Spotify and focuses on building RESTful APIs.

## Features

- User Registration and Login
- JWT Authentication
- Password Hashing using Bcrypt
- MongoDB Integration
- Protected Routes
- Middleware Implementation
- Error Handling
- REST API Architecture

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Bcrypt
- Postman


## What I Learned

- Backend Development Fundamentals
- REST API Design
- Authentication & Authorization
- MongoDB CRUD Operations
- Middleware Usage
- Git & GitHub Basics
